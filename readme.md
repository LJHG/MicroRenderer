# MicroRenderer

A self-built soft Rasterizer, called Micro(soft)Renderer.

## process
- [x] Triangle Rasterization(line mode and fill mode)
- [x] linear and barycentric interpolation
- [x] Z-buffering
- [x] mvp transformation


## dependencies
1. SDL2
2. stb_image
3. glm
